# EcoAir Plastics Clone

An AI-powered clone of [EcoAir Plastics](https://ecoairplastics.com/) built with **Astro** and **Tailwind CSS**.

## 🚀 Tech Stack

- **[Astro](https://astro.build/)** - The web framework for content-driven websites
- **[Tailwind CSS](https://tailwindcss.com/)** - A utility-first CSS framework
- **AI-Assisted Development** - Built entirely with AI tools for rapid development
